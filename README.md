# 🌲 Biovitro App: Phygital Focus & Sustainability

**Biovitro App** é um ecossistema de produtividade inspirado na técnica Pomodoro e no conceito "Phygital" (Físico + Digital). O objetivo é ajudar usuários a vencer a procrastinação enquanto promovem o reflorestamento real e virtual.

---

## 🚀 O Diferencial: Experiência Phygital
Diferente de outros apps de foco, o **Rooted** integra um **Kit de Cultivo Físico**.
- **QR Code Único:** Cada kit físico contém um token criptográfico em um QR Code.
- **Validação de Uso Único:** Ao escanear, o app valida o token no backend, desbloqueando árvores lendárias ou convertendo em moedas virtuais caso o usuário já possua o item.
- **Conexão Real:** Incentiva o usuário a plantar na vida real o que cultiva no digital.

| **Scan de Recompensa** | Integração com câmera para resgate de itens via kits físicos. |

## 🏗️ Arquitetura e Tecnologias

Este projeto foi desenvolvido aplicando conceitos de **Engenharia de Software** e **Clean Architecture** para garantir escalabilidade e manutenção.

- **Frontend:** [React Native / Flutter] - *Foco em UX fluida e acesso à câmera.*
- **Backend:** [Node.js / Python Fast API] - *Gerenciamento de tokens de QR Code e lógica de Gacha.*
- **Banco de Dados:** [PostgreSQL / MongoDB] - *Persistência de inventário e logs de foco.*
- **Segurança:** Validação de tokens via [JWT/HMAC] para evitar fraudes em códigos físicos.

### 🧠 Módulo de Inteligência Vegetal (IA)
O Rooted utiliza **Computer Vision** para auxiliar no cuidado botânico:
- **Identificação Automática:** Reconhecimento de espécies via câmera utilizando modelos de Deep Learning.
- **Guia de Cultivo Personalizado:** Instruções específicas de rega e iluminação integradas ao sistema **BioVitro**.
- **Diagnóstico de Saúde:** Identificação de pragas e doenças comuns com sugestões de tratamento orgânico e químico (adubação líquida vs. pó).
